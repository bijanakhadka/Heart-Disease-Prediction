# Heart Disease Prediction

A machine learning project that predicts the likelihood of heart disease using the Cleveland Heart Disease Dataset.

This repository includes a complete data science workflow covering data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model evaluation, and a Streamlit web application for interactive predictions.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Jupyter Notebook
